# 文字江湖 (MUD Demo)

这是一个用 C 语言编写的简约文字 MUD 游戏骨架。

## 核心功能

- **look**：查看当前房间描述及出口。
- **go [direction]**：在房间间移动（支持 north, south, east, west）。
- **status**：查看玩家当前的生命值、攻击力和防御力。
- **help**：显示指令帮助列表。
- **quit**：退出游戏。

## 快速开始

### 编译
本项目包含一个 `main.c` 文件，可以使用 gcc 进行编译：

```bash
gcc main.c -o mud.exe
```

### 运行
```bash
./mud.exe
```

## 项目特色
- **跨平台兼容**：在 Windows 下支持自动切换代码页（UTF-8），解决中文乱码问题。
- **ANSI 颜色**：控制台输出带有色彩高亮，视觉体验极佳。
- **KISS 原则**：代码结构简单直观，适合作为 MUD 游戏开发的入门参考。
